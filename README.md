# 1. 프로젝트 목적
+ 소비자의 필기구 구매에 도움이 되도록 제품의 특성을 분류하여 제공하는 서비스
+ 워드클라우드를 활용하여 제품의 후기 파악이 더욱 직관화되도록 제공
+ 웹 사이트의 접근환경이 다양화되고 있는 점을 고려, Bootstrap을 활용한 유동성 페이지 구성하여 다양한 기기에서 안정적인 접속
